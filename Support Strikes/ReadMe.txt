We're [url=gitlink]Open Source![/url]

This is a Support Strike mod, inspired from several games (same name in Metal Gear Solid V: The Phantom Pain, Tatical aid in World in Conflict, etc.) where you can call in strike packages to destroy units on the field, or help the player gain an advantage over the enemy. This mod attempts to bridge that gap by implementing some Support Strikes while fitting it within XCom's gameplay.

This mod only has 3 Support Strikes implemeted so far: Mortar Strikes, A-10 Strafing Run, and the Ion Cannon. There were gonna be more but I just don't have the time or motivation anymore, so I decided to release what I have so far. Right now it works without much issues on a vanilla playthrough. Heavily modded playthroughs may have issues.

If you have issues with this mod or spotted a bug, please report it in the [url=https://steamcommunity.com/workshop/filedetails/discussion/2805796212/3270186971450162379/]Bug Report Thread[/url]. If you want to provide feedback, whether its good or bad, then post in the [url=https://steamcommunity.com/workshop/filedetails/discussion/2805796212/3270186971450164226/]General Feedback Thread[/url].

If you are thinking of downvoting this mod, please provide feedback first, so I know what I should improve on/did wrong.

[h1][b]HOW IT WORKS[/b][/h1]
[h1]-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.-.[/h1]

Support Strikes allows you to alter the battlefield in a significant way, but there are several drawbacks that you will need to consider when using them.
[list]
	[*]Most Strikes require open skies. Underground or alien structures that are enclosed do not allow any sort of support strikes.
	[*]Support Strikes takes time to arrive to the battlefield, so you will have to time your strikes appropriately.
	[*]All Support Strikes cost Intel to perform, and gets more expensive the more times you call in strikes. The costs reset every month.
	[*]Mortars saturate an area with explosives, but are hardly accurate.
	[*]Smoke Mortars can provide a standard defense boost in a large area.
	[*]A10 Strafing Run is very accurate but costs more intel and requires the aircraft arrive in the AO before it can be used. Strafing Runs can be used once per turn, up to 3 turns, before needing to return to rearm and refuel.
	[*]The Ion Cannon is extremely accurate and quick but requires a lot of resources to invest into, and uses an extreme amount of intel.
[/list]
